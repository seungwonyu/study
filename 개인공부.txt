9. 배열의 고차함수와 내장함수의 개념을 설명할 수 있다.
고차함수는 함수의 형태로 리턴할수 있는 함수  (Higher-Order Function)
함수를 인자로 전달받거나 연산의 결과로 반환해주는 메서드
함수를 연산의 결과로 반환하는것은 몇가지 장점이 있는데, 이떄 장점이 변수를 보호 할 수 있는것 
( 함수안에서 선언이 된 변수는 함수 밖에서 사용할 수 없지만 내부함수에서는 부모함수의 변수를 이어 받아 사용가능하기 때문에 사용 가능.)
한번에 여러 변수를 내부에서 선언 하고 활용할 수 있다는것
클로저 기능을 사용할 수 있다는것이다.
클로저 기능이란 무엇인가? >> 
function init() {
  var name = "Mozilla"; // name은 init에 의해 생성된 지역 변수이다.
  function displayName() {
    // displayName() 은 내부 함수이며, 클로저다.
    console.log(name); // 부모 함수에서 선언된 변수를 사용한다.
  }
  displayName();
}
init();이 함수는 내부함수가 존재하며 리턴값이 함수인 init이란 함수이다.
이때 내부함수인 displayName은 init의 name변수에 접근이 가능하다.
이는 함수가 중첩될때 변수이름을 확인하는 방법을 설명하는 '정적스코프'의 예시이다.
이는 변수가 어디에서 사용 가능한지 알기위해 소스코드 내 어디에서 선언되었는지를 고려하는것
https://poiemaweb.com/js-closure > 클로저에 대한 공부 하기 좋게 정리가 잘 되어있다.
결국 클로저란 내가 사용할 변수의 무작위 이용을 막거나 힐떼 주로 사용이 되는것 같다.
즉 변수의 사용에 매우 효과적인 듯
